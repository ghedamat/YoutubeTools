== YoutubeTools

* http://github.com/maykon/YoutubeTools

== DESCRIPTION

Gem from search, download and converter youtube videos to mp3

== Dependencies

* ruby 1.8.6
* hpricot[http://github.com/hpricot]
* open-uri

== Installation

Install the gem:

  gem install "youtube_tools"

Add it to your Gemfile:

  gem "youtube_tools"
